Aprendendo Github

Repositório é uma área onde o código está sendo versionado

  • COMMIT

    •    ele atualiza o repositorio local

  • PUSH

    •    sobe para o repositorio remoto (github)

  • PULL

    •    trás do repositorio remoto para o local (tipo baixar do github)

  • "U" de Untracked

    •    quer dizer que as mudanças feitas no visual code não foram monitoradas pelo Github

  • "M" de Modified

    •    as mudanças foram detectadas
